# Environment Variable Reference

Status: production and local/test reference.

## Backend

Required in all environments:

- `CONTEXTOS_DATABASE_URL`: async SQLAlchemy PostgreSQL URL for runtime access.
- `CONTEXTOS_REDIS_URL`: Redis URL used by the API and worker.

Required in production:

- `CONTEXTOS_ENVIRONMENT=production`
- `CONTEXTOS_LOG_FORMAT=json`
- `CONTEXTOS_DATABASE_URL`: non-local production runtime database URL.
- `CONTEXTOS_MIGRATION_DATABASE_URL`: non-local migration database URL for Alembic.
- `CONTEXTOS_REDIS_URL`: non-local Redis URL.
- `CONTEXTOS_SUPABASE_URL`: exact HTTPS Supabase project URL.
- `CONTEXTOS_SUPABASE_JWT_ISSUER`: exact HTTPS issuer URL.
- `CONTEXTOS_SUPABASE_JWKS_URL`: exact HTTPS JWKS URL.
- `CONTEXTOS_SUPABASE_SECRET_KEY`: server-side Supabase secret key.
- `CONTEXTOS_FRONTEND_URL`: exact HTTPS frontend URL.
- `CONTEXTOS_DOCUMENT_STORAGE_BACKEND=supabase`
- `CONTEXTOS_SUPABASE_STORAGE_BUCKET`: private bucket for uploaded PDFs.
- `CONTEXTOS_LLM_PROVIDER=gemini`
- `CONTEXTOS_EMBEDDING_PROVIDER=gemini`
- `CONTEXTOS_AI_PROVIDER_API_KEY`: server-side provider key.

Production startup fails if local/test database or Redis URLs are used, if fake AI providers are selected, if required secrets are blank, or if Auth/frontend URLs are not exact HTTPS URLs.

Optional backend settings:

- `CONTEXTOS_APPLICATION_NAME`
- `CONTEXTOS_APPLICATION_VERSION`
- `CONTEXTOS_LOG_LEVEL`
- `CONTEXTOS_READINESS_TIMEOUT_SECONDS`
- `CONTEXTOS_SUPABASE_JWT_AUDIENCE`
- `CONTEXTOS_SUPABASE_ALLOWED_JWT_ALGORITHMS`
- `CONTEXTOS_SUPABASE_JWKS_CACHE_TTL_SECONDS`
- `CONTEXTOS_SUPABASE_JWT_CLOCK_SKEW_SECONDS`
- `CONTEXTOS_BETA_MAX_USERS`
- `CONTEXTOS_DOCUMENT_STORAGE_ROOT`: local/test storage only.
- `CONTEXTOS_SUPABASE_STORAGE_PATH_PREFIX`
- `CONTEXTOS_DOCUMENT_MAX_PER_USER`
- `CONTEXTOS_DOCUMENT_MAX_PDF_SIZE_BYTES`
- `CONTEXTOS_DOCUMENT_MAX_TOTAL_SIZE_BYTES`
- `CONTEXTOS_DOCUMENT_MAX_PAGES`
- `CONTEXTOS_DOCUMENT_CHUNK_SIZE`
- `CONTEXTOS_DOCUMENT_CHUNK_OVERLAP`
- `CONTEXTOS_DOCUMENT_QUEUE_NAME`
- `CONTEXTOS_RETRIEVAL_TOP_K`
- `CONTEXTOS_RETRIEVAL_SIMILARITY_THRESHOLD`
- `CONTEXTOS_RETRIEVAL_MAX_CONTEXT_CHARACTERS`
- `CONTEXTOS_MEMORY_RETRIEVAL_TOP_K`
- `CONTEXTOS_MEMORY_RETRIEVAL_SIMILARITY_THRESHOLD`
- `CONTEXTOS_MEMORY_RETRIEVAL_MAX_CONTEXT_CHARACTERS`
- `CONTEXTOS_LLM_MODEL`
- `CONTEXTOS_EMBEDDING_MODEL`
- `CONTEXTOS_EMBEDDING_DIMENSION`
- `CONTEXTOS_AI_PROVIDER_TIMEOUT_SECONDS`
- `CONTEXTOS_AI_PROVIDER_MAX_RETRIES`
- `CONTEXTOS_AI_DAILY_MESSAGE_LIMIT`
- `CONTEXTOS_AI_MONTHLY_MESSAGE_LIMIT`

## Frontend

Frontend-safe variables:

- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_PUBLISHABLE_KEY`
- `NEXT_PUBLIC_SITE_URL`
- `CONTEXTOS_API_URL`

Never expose `CONTEXTOS_SUPABASE_SECRET_KEY`, `CONTEXTOS_AI_PROVIDER_API_KEY`, database URLs, Redis URLs, signed URLs, or storage service credentials to the frontend.

## Local And Test

Local and test environments may use local PostgreSQL, Redis, local file storage, and fake AI providers. Keep these values out of production service settings.
